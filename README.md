# 🧰 WinToolBox - Suite d’administration Windows

## 🎯 Description
WinToolBox est une suite d’outils d’administration Windows développée entièrement en **Batch (Shell Windows)**.  
Elle permet de réaliser des opérations de maintenance, de sécurité et de gestion système depuis une interface CLI interactive.

## ⚙️ Modules inclus
1. Informations système  
2. Nettoyage & optimisation  
3. Surveillance réseau  
4. Gestion des utilisateurs  
5. Sécurité & pare-feu  
6. Sauvegarde & restauration  

## 📁 Structure du projet
WinToolBox/
├── WinToolBox.bat
├── modules/
└── logs/

## 🚀 Utilisation
1. Exécute `WinToolBox.bat` en tant qu’administrateur.  
2. Sélectionne un module depuis le menu.  
3. Consulte les rapports dans le dossier `logs/`.
